Nomad

Nomad is a modern, multi-paradigm language designed to be simple. Nomad is inspired by ML and TypeScript and simplifies the concepts of these languages, so new features can be added.

Syntax

Nomad is simple and designed to be approachable. It supports both an ML style syntax: 

				double a: Float -> Float 
				double a = a + a

As well as a C style syntax

				double(a: Float): Float => a + a
